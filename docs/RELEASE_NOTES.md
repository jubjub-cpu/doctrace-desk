# Release Notes

## v1.0.0

DocuTrace Desk's first standalone release provides a complete synthetic document-review workflow.

### Included

- Two fictional review packets with six source documents
- Focused evidence search with transparent query expansion
- Line-level citations and source inspector
- Citation verification controls
- Seven structured fields with source references
- Clause-level baseline and proposed-version comparison
- Risk flags tied to changed commercial and operational terms
- Human approve or return gate
- Local copy and text-download summary
- Loading, error, validation, empty, pending, approved, and returned states
- Responsive desktop and mobile layouts
- Logic tests, repository validation, privacy scan, and browser workflow evidence
- Architecture, case study, validation report, privacy notes, and runtime boundaries

### Boundaries

- Deterministic browser analysis, not a live model
- Fictional data only
- Not legal advice or compliance certification
- No external action, backend, persistence, or authentication
- No paid API or sign-in required
